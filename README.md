# 🌌 Ontological Void

A minimalist, deep-purple workspace powered by **Gentoo Linux**. Focused on efficiency, aesthetic consistency, and anarchist ontology.

---

## ⚙️ The Stack

* **OS:** [Gentoo](https://www.gentoo.org/) (Zen Kernel)
* **WM:** [i3-wm](https://i3wm.org/)
* **Bar:** [Polybar](https://github.com/polybar/polybar)
* **Terminal:** [Alacritty](https://alacritty.org/)
* **Editor:** [Doom Emacs](https://github.com/doomemacs/doomemacs)
* **Font:** JetBrainsMono Nerd Font
* **Visuals:** btop, cava, picom

## 📂 Structure

- `i3/`: Window manager logic and keybindings.
- `polybar/`: Custom purple modules and bar layout.
- `picom/`: Animation and transparency rules.
- `rofi/`: App launcher and wall selector.
